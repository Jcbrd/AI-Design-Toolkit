# 🎨 AI Design Toolkit

A collection of Claude skills for product designers working on AI-powered products — built to help you plan faster, design smarter, and ship with more confidence.

---

## What's in here

| Skill | What it does |
|---|---|
| [design-roadmap-architect](./design-roadmap-architect/) | Turn a project idea into a framework-anchored roadmap with personalized tool picks |
| [ai-ux-patterns](./ai-ux-patterns/) | Map an AI feature to the patterns that matter for it, ranked by criticality |

---

## How to use these skills

These are Claude skills — they run inside [Claude](https://claude.ai) and give it specialized knowledge for design work. To use one:

1. Open Claude
2. Start a new conversation
3. Paste the contents of the skill's `SKILL.md` file into the system prompt
4. Describe your project or feature and go

Each skill's folder has its own README with details on what it does, when to use it, and what you get back.

---

## Who this is for

Product designers and UX designers who are:

- Working on AI-powered features or products
- Trying to figure out where to start on a new project
- Looking for more structure than vibes, less overhead than a full design system

No engineering background required. These skills are written in plain language and assume Figma-level tool fluency, not code.
